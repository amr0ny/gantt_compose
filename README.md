# Проект по управлению проектами с использованием диаграмм Ганта

## Описание

Этот сервис предоставляет возможности для управления проектами, включая создание диаграмм Ганта, трекинг задач, отслеживание статуса задач, редактирование в календаре или таблице, работу с командой, установку ролей в проекте и назначение ответственных за задачи.

## Функционал и возможности

- **Создание диаграмм Ганта**: Визуализация проекта и задач на диаграмме Ганта.
- **Трекинг задач**: Отслеживание прогресса задач и их статуса.
- **Редактирование задач**: Возможность редактировать задачи в календаре или таблице.
- **Работа с командой**: Совместное управление проектами, назначение задач членам команды.
- **Установка ролей**: Настройка ролей для пользователей в проекте и назначение ответственных за задачи.
- **Управление проектами**: Полный контроль над своими проектами и задачами.
